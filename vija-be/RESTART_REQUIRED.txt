========================================
⚠️  RESTART BACKEND REQUIRED
========================================

Dashboard query đã được cập nhật để tách riêng từng PO.

Thay đổi:
- Trước: Gộp các Mã PO lại khi trùng Mã BV (GROUP_CONCAT)
- Sau: Tách riêng từng dòng cho mỗi cặp (Mã PO, Mã BV)

Ví dụ:
Trước: 1 dòng - BG001, "PO001,PO002", BV001
Sau:   2 dòng - BG001, PO001, BV001
              - BG001, PO002, BV001

Cách restart:
1. Dừng backend (Ctrl + C)
2. Chạy lại: npm run dev
3. Refresh trang Dashboard
